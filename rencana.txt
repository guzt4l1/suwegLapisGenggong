3️⃣ Strategi Aman Biar Tidak Cepat Habis Jatah
Optimasi gambar → simpan di layanan gratis seperti Cloudinary atau Imgur.

Gunakan caching → untuk API Node.js, cache response statis.

Buat WordPress dalam versi demo ringan (nonaktifkan plugin berat).

Bagi traffic → taruh beberapa project di GitHub Pages supaya beban Vercel/Netlify nggak terlalu besar.

Gunakan link preview (misal video YouTube) untuk project yang resource-nya berat, sehingga orang hanya akses full site kalau mau lihat detail.
